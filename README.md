- **인터랙티브 발표**: [`viewer.html`](./viewer.html) — 영상 인라인 재생 가능

## 스택

- 슬라이드: HTML/CSS (`slides-grab` 프레임워크)
- 폰트: Pretendard + JetBrains Mono
- 스타일: modern-dark (모노크롬, 악센트 없음)
- 영상: HTML5 `<video>` 인라인 재생
- 자동화: `_gen.py` (Python 슬라이드 생성기)
